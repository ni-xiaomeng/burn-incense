# 一炷香 · 电子烧香

> 极简的电子烧香小应用：一支线香，点击点燃，30 分钟燃尽，柔和烟雾。

## 在线体验

部署到 GitHub Pages 后访问：
`https://<你的GitHub用户名>.github.io/<仓库名>/`

## 本地使用

直接双击 `index.html` 用浏览器打开即可，无需任何依赖。

## 操作

- 进入页面后，**点击屏幕** 即点燃这支香
- 顶部显示 30 分钟倒计时，香体随时间逐渐缩短
- 30 分钟后燃尽，提示「香 已 燃 尽」

## 文件

- `index.html` — 应用全部内容（HTML + CSS + JS 单文件，无外部依赖）
